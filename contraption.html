Online Contraption WebGL interface

Project Description

Previous research and user evaluation concluded that having a tangible user interface for the design of room layouts with daylighting simulations helped user gain insight into how to better provide natural lighting for spaces with to create energy efficient buildings. In the design process most 3D modeling software requires a complete model to run lighting simulations on, however our tool is quick and easy much earlier in the design phase. This allows for the consideration of lighting much sooner and in response architects can better plan their rooms to maximize the use of natural lighting in spaces For research this past summer I offer a web-based interface that allows for lighting visualization while building a models. The following website is prototype that I use to gather information on how long it takes users to create, render, and renovates models.

Project Details

  • Uses WebGL and three.js library to render 3D visualization on web browsers
  • Uses same key algorithms from the tangible user interface
  • Uses photon mapping to get global illumination to generate lighting textures
  • Runs a machine with a GTX 500 series NVIDIA Graphics Card
  • Saves statistical significant information for each model created
